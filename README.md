# HITS Smart Campus - Red Team Assessment

## Project Overview

This repository contains my contribution to the HITS Smart Campus Red Team Assessment project, completed as part of the Masai School Cybersecurity Program.

The project simulates a Vulnerability Assessment and Penetration Testing (VAPT) engagement for a smart campus environment. The objective is to identify security vulnerabilities, assess their potential impact, and recommend appropriate remediation measures.

## My Contribution

My responsibilities in this project include:

- Analyzing identified vulnerabilities
- Assessing their severity and business impact
- Preparing risk ratings
- Recommending remediation measures
- Documenting the findings
